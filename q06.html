문장 1

The question “Where does happiness come from?” asks whether happiness comes from outside conditions or inner ways of living.

“행복은 어디에서 오는가?”라는 질문은 행복이 외부 조건에서 오는지, 아니면 내면의 삶의 방식에서 오는지를 묻는다.

핵심 문법: class, dict, key, value
학습 목표: 문장 정보를 딕셔너리로 저장하고 클래스에서 관리합니다.

OOP 파이썬 코드

class SentenceCard:
    def __init__(self, english, korean):
        self.data = {
            "english": english,
            "korean": korean,
            "topic": "happiness"
        }

    def show(self):
        print(self.data["english"][:30])
        print(self.data["korean"][:20])

card = SentenceCard('The question “Where does happiness come from?” asks whether happiness comes from outside conditions or inner ways of living.', '“행복은 어디에서 오는가?”라는 질문은 행복이 외부 조건에서 오는지, 아니면 내면의 삶의 방식에서 오는지를 묻는다.')
card.show()

class DictTransformer:
    def __init__(self, text):
        self.data = {"original": text}

    def transform(self):
        upper = lambda s: s[:25].upper()
        self.data["upper_front"] = upper(self.data["original"])
        return self.data

obj = DictTransformer('In humanistic philosophy, happiness is not only pleasure, but a deep harmony of the soul, relationships, and purpose.')
print(obj.transform())

class WhileDictReader:
    def __init__(self, english, korean):
        self.data = {"english": english, "korean": korean, "theme": "happiness"}

    def read_keys(self):
        keys = list(self.data.keys())
        i = 0
        while i < len(keys):
            key = keys[i]
            print(key, ":", str(self.data[key])[:20])
            i += 1

obj = WhileDictReader('Confucius saw happiness in harmonious relationships and a life guided by respect.', '공자는 조화로운 관계와 존중으로 이끌리는 삶 속에서 행복을 보았다.')

OOP 파이썬 코드

class MultiDict:
    def collect(self, *dicts):
        result = {}
        for d in dicts:
            result.update(d)
        return result

a = {"en": 'For example, a warm conversation with family can give deeper happiness than expensive possessions.'}
b = {"ko": '예를 들어 가족과의 따뜻한 대화는 값비싼 소유물보다 더 깊은 행복을 줄 수 있다.'}
obj = MultiDict()
print(obj.collect(a, b))

class KwargsDict:
    def __init__(self, **kwargs):
        self.data = kwargs

    def summary(self):
        return {
            "keys": list(self.data.keys()),
            "english_front": self.data["english"][:25]
        }

I will enjoy simple things, such as light, food, conversation, and rest.

나는 빛, 음식, 대화, 휴식 같은 단순한 것들을 누리겠다.

핵심 문법: sort by value length
학습 목표: 딕셔너리 값을 길이 기준으로 정렬합니다.

OOP 파이썬 코드

class ValueLengthSorter:
    def __init__(self, *sentences):
        self.data = {f"s{i}": s for i, s in enumerate(sentences, start=1)}

    def sort_by_length(self):
        return dict(sorted(self.data.items(), key=lambda item: len(item[1])))

obj = ValueLengthSorter(
    'Happiness comes from a heart that notices kindness and meaning.',
    'I will speak warmly to my students and people around me.',
    'I will enjoy simple things, such as light, food, conversation, and rest.'
)
print(obj.sort_by_length())

상세 풀이

이 예제는 문장 정보를 dict로 저장하고, class 안에서 관리하는 OOP 방식입니다. 딕셔너리는 {"key": value} 구조로 데이터를 저장하며, items(), keys(), pop(), update(), dict(zip()), 딕셔너리 컴프리헨션 등을 통해 데이터를 체계적으로 처리할 수 있습니다. 또한 함수, 람다, join, zip, map, for, if, while, *args, **kwargs, filter, index, sorted를 함께 사용하여 실전적인 데이터 처리 구조를 익힙니다.

OOP 파이썬 코드

class BilingualDict:
    def __init__(self, english, korean):
        self.data = {
            "english": {"text": english, "front": english[:25]},
            "korean": {"text": korean, "front": korean[:15]}
        }

    def show(self):
        for lang, info in self.data.items():
            print(lang, "=>", info["front"])

obj = BilingualDict('I will help someone today, because shared goodness brings deep joy.', '나누어진 선함은 깊은 기쁨을 가져오므로 나는 오늘 누군가를 돕겠다.')

핵심 문법: search, filter, dict
학습 목표: 특정 단어가 들어간 문장을 딕셔너리에 저장합니다.

OOP 파이썬 코드

class SearchDict:
    def __init__(self, *sentences):
        self.data = {i: s for i, s in enumerate(sentences, start=1)}

    def search(self, keyword):
        return dict(filter(lambda item: keyword.lower() in item[1].lower(), self.data.items()))

obj = SearchDict(
    'I will enjoy simple things, such as light, food, conversation, and rest.',
    'When I feel tired, I will remember what I am thankful for.',
    'I will help someone today, because shared goodness brings deep joy.',
    'I will avoid comparison and walk my own meaningful path.'
)
print(obj.search("happiness"))

감사와 사랑으로 나는 오늘을 더 행복하고 의미 있게 만들 수 있다.

핵심 문법: class, *args, **kwargs
학습 목표: 클래스 안에서 가변 인자와 옵션을 활용해 분석합니다.

OOP 파이썬 코드

class OOPAnalyzer:
    def __init__(self, *sentences, **options):
        self.sentences = sentences
        self.options = options
        self.data = {}

    def analyze(self):
        start = self.options.get("start", 0)
        end = self.options.get("end", 20)
        for i, sentence in enumerate(self.sentences, start=1):
            self.data[i] = sentence[start:end]
        return self.data

obj = OOPAnalyzer('I will avoid comparison and walk my own meaningful path.', 'With gratitude and love, I can make today happier and more meaningful.', start=0, end=30)
print(obj.analyze())

문장 28

최종 OOP 딕셔너리 미션

Today, I will live as a person who finds happiness in wisdom, kindness, and purpose.

오늘 나는 지혜와 친절과 목적 속에서 행복을 찾는 사람으로 살아가겠다.

핵심 문법: OOP, filter, sorted, dict
학습 목표: 필터, 정렬, 딕셔너리, 클래스를 종합합니다.

OOP 파이썬 코드

class FinalDictMission:
    def __init__(self, *sentences):
        self.data = {i: s for i, s in enumerate(sentences, start=1)}

    def run(self, keyword="happiness"):
        filtered = dict(filter(lambda item: keyword.lower() in item[1].lower(), self.data.items()))
        sorted_items = sorted(filtered.items(), key=lambda item: len(item[1]))
        result = {}
        for index, sentence in sorted_items:
            result[index] = {
                "front": sentence[:25],
                "back": sentence[-20:],
                "length": len(sentence)
            }
        return result

obj = FinalDictMission(
    'I will avoid comparison and walk my own meaningful path.',
    'With gratitude and love, I can make today happier and more meaningful.',
    'Today, I will live as a person who finds happiness in wisdom, kindness, and purpose.'
)
print(obj.run())

상세 풀이

이 예제는 문장 정보를 dict로 저장하고, class 안에서 관리하는 OOP 방식입니다. 딕셔너리는 {"key": value} 구조로 데이터를 저장하며, items(), keys(), pop(), update(), dict(zip()), 딕셔너리 컴프리헨션 등을 통해 데이터를 체계적으로 처리할 수 있습니다. 또한 함수, 람다, join, zip, map, for, if, while, *args, **kwargs, filter, index, sorted를 함께 사용하여 실전적인 데이터 처리 구조를 익힙니다.
