각 문장을 단어 리스트로 바꾸어 함수, 람다, join, zip, map, pop, for, if, while, *args, **kwargs, filter, 인덱스, 정렬 등을 활용한 리스트 슬라이싱을 다양하게 연습합니다.

문장 1

함수로 리스트 앞뒤 요소 추출

The question “What is truth?” is one of the starting points of philosophy.

“무엇이 진실인가?”라는 질문은 철학의 출발점 중 하나이다.

핵심 문법: function, list index, list slicing
학습 목표: 문장을 단어 리스트로 바꾸고 첫 단어, 마지막 단어, 앞 단어들을 추출합니다.

파이썬 코드

sentence = 'The question “What is truth?” is one of the starting points of philosophy.'
words = sentence.split()

def list_edges(items):
    first = items[0]
    last = items[-1]
    front = items[:5]
    return first, last, front

a, b, c = list_edges(words)
print("첫 단어:", a)
print("마지막 단어:", b)
print("앞 5개 단어:", c)

상세 풀이

이 예제는 문장을 split()으로 단어 리스트로 만든 뒤 리스트 슬라이싱을 연습합니다. 리스트는 0번부터 인덱스가 시작하며, words[start:end:step] 형식으로 앞부분, 뒷부분, 일정 간격, 역순을 추출할 수 있습니다. 함수, 람다, join, zip, map, pop, for, if, while, *args, **kwargs, filter, index, sorted를 함께 사용해 리스트 처리 능력을 키웁니다.

    'A judge seeks truth by listening to many sides before making a decision.',
    'A teacher seeks truth by admitting when new evidence corrects an old belief.'
)

문장 11

**kwargs로 슬라이싱 범위 지정

Immanuel Kant reminded us that human understanding has limits and structures.

임마누엘 칸트는 인간의 이해에는 한계와 구조가 있음을 일깨워 주었다.

학습 목표: 키워드 인자로 시작과 끝을 받아 단어 리스트를 자릅니다.

파이썬 코드

def slice_words(**kwargs):
    words = kwargs["text"].split()
    start = kwargs.get("start", 0)
    end = kwargs.get("end", 5)
    return words[start:end]

print(slice_words(text='Immanuel Kant reminded us that human understanding has limits and structures.', start=2, end=8))

파이썬 코드

def transform_words(*sentences, **kwargs):
    start = kwargs.get("start", 0)
    end = kwargs.get("end", 5)
    reverse = kwargs.get("reverse", False)
    result = []
    for sentence in sentences:
        words = sentence.split()[start:end]
        if reverse:
            words = words[::-1]
        result.append(words)
    return result

print(transform_words('I will avoid gossip, exaggeration, and careless judgment.', 'With truth and humility, I can make today more meaningful.', start=0, end=6, reverse=True))

    'I will avoid gossip, exaggeration, and careless judgment.',
    'With truth and humility, I can make today more meaningful.',
    'Today, I will live as a person who loves truth and practices honesty.'
]

def final_mission(*sentences, **kwargs):
    keyword = kwargs.get("keyword", "truth")
    selected = list(filter(lambda s: keyword.lower() in s.lower(), sentences))
    selected = sorted(selected, key=lambda s: len(s.split()))
    for index, sentence in enumerate(selected, start=1):
        words = sentence.split()
        print(index, words[:4], "...", words[-3:])

final_mission(*sentences, keyword="truth")
