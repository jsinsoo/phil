어떻게 살아야 하는가? 문장 기반 파이썬 문자열 슬라이싱 학습

각 문장을 소재로 함수, 람다, join, zip, map, pop, for, if, while, *args, **kwargs, filter, 인덱스, 정렬 등을 활용하여 문자열 슬라이싱을 다양하게 연습합니다.

점수: 아직 채점하지 않음

채점 후 TXT 저장을 누르면 학번_이름_02_점수_응시시각.txt 형식으로 저장됩니다.

문장 1

함수로 앞뒤 문자 추출

The question “How should I live?” is the heart of ethics and practical philosophy.

“어떻게 살아야 하는가?”라는 질문은 윤리학과 실천철학의 핵심이다.

핵심 문법: function, index, slicing
학습 목표: 함수를 만들어 문자열의 첫 글자, 마지막 글자, 앞부분을 추출합니다.

파이썬 코드

sentence = 'The question “How should I live?” is the heart of ethics and practical philosophy.'

def slice_edges(text):
    first = text[0]
    last = text[-1]
    front = text[:15]
    return first, last, front

a, b, c = slice_edges(sentence)
print("첫 글자:", a)
print("마지막 글자:", b)
print("앞 15글자:", c)

상세 풀이

이 코드는 위 문장을 하나의 문자열 자료로 다룹니다. 문자열은 왼쪽부터 0, 1, 2처럼 인덱스가 붙고, text[start:end:step] 형식으로 원하는 부분을 잘라낼 수 있습니다. 함수, 람다, join, zip, map, pop, for, if, while, *args, **kwargs, filter, 인덱스, 정렬을 함께 사용하면 긴 문장도 구조적으로 분석할 수 있습니다.

확인 문제

Q1. 1번 예제에서 text[:5]의 의미는?

정답: 1번
text[:5]는 0번 인덱스부터 5번 직전까지, 즉 앞 5글자를 가져옵니다.

**kwargs 선택 슬라이싱

니체는 물려받은 규칙만으로 살지 말고 용기 있게 가치를 창조하라고 도전했다.

핵심 문법: **kwargs, keyword arguments
학습 목표: start와 end 값을 키워드 인자로 받아 원하는 범위를 자릅니다.

파이썬 코드

def cut_text(**kwargs):
    text = kwargs["text"]
    start = kwargs.get("start", 0)
    end = kwargs.get("end", 20)
    return text[start:end]

print(cut_text(text='Nietzsche challenged people to create values with courage instead of living only by inherited rules.', start=5, end=28))

학습 목표: 여러 문장을 받아 옵션에 따라 대문자 변환까지 처리합니다.

파이썬 코드

def transform(*args, **kwargs):
    start = kwargs.get("start", 0)
    end = kwargs.get("end", 15)
    upper = kwargs.get("upper", False)
    result = []
    for text in args:
        piece = text[start:end]
        if upper:
            piece = piece.upper()
        result.append(piece)
    return result

print(transform('I will reflect on my choices and try to become a better person.', 'With love and responsibility, an ordinary day can become a good life.', start=0, end=18, upper=True))

파이썬 코드

sentences = [
    'I will reflect on my choices and try to become a better person.',
    'With love and responsibility, an ordinary day can become a good life.',
    'Today, I will live truthfully, warmly, and meaningfully.'
]

def final_mission(*args, **kwargs):
    keyword = kwargs.get("keyword", "live")
    selected = list(filter(lambda s: keyword.lower() in s.lower(), args))
    selected = sorted(selected, key=len)
    for idx, sentence in enumerate(selected, start=1):
        print(idx, sentence[:22], "...", sentence[-10:])

final_mission(*sentences, keyword="live")
