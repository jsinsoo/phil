05 파이썬 문자열 슬라이싱

죽음 이후는 무엇인가? 문장 기반 파이썬 문자열 슬라이싱 학습

각 문장을 소재로 함수, 람다, join, zip, map, pop, for, if, while, *args, **kwargs, filter, 인덱스, 정렬 등을 활용한 문자열 슬라이싱을 다양하게 연습합니다.

점수: 아직 채점하지 않음

채점 후 TXT 저장을 누르면 학번_이름_05_점수_응시시각.txt 형식으로 저장됩니다.

문장 1

함수로 문자열 앞뒤 문자 추출

The question “What comes after death?” has been asked by religions, philosophies, and ordinary people for thousands of years.

“죽음 이후는 무엇인가?”라는 질문은 수천 년 동안 종교와 철학, 그리고 평범한 사람들이 물어 온 질문이다.

핵심 문법: function, index, string slicing
학습 목표: 함수로 첫 글자, 마지막 글자, 앞부분을 추출합니다.

파이썬 코드

sentence = 'The question “What comes after death?” has been asked by religions, philosophies, and ordinary people for thousands of years.'

def string_edges(text):
    first = text[0]
    last = text[-1]
    front = text[:20]
    return first, last, front

a, b, c = string_edges(sentence)
print("첫 글자:", a)
print("마지막 글자:", b)
print("앞 20글자:", c)

상세 풀이

이 예제는 위 문장을 문자열 자료로 저장한 뒤 문자열 슬라이싱을 연습합니다. 문자열은 0번부터 인덱스가 시작하며, text[start:end:step] 형식으로 앞부분, 뒷부분, 일정 간격, 역순을 추출할 수 있습니다. 함수, 람다, join, zip, map, pop, for, if, while, *args, **kwargs, filter, index, sorted를 함께 사용해 문자열 처리 능력을 키웁니다.

문장 11

**kwargs 슬라이싱 범위 지정

어떤 전통들은 천국, 환생, 혹은 영혼의 불멸을 믿는다.

핵심 문법: **kwargs, start, end
학습 목표: 키워드 인자로 시작과 끝 범위를 받아 문자열을 자릅니다.

파이썬 코드

def cut_text(**kwargs):
    text = kwargs["text"]
    start = kwargs.get("start", 0)
    end = kwargs.get("end", 25)
    return text[start:end]

print(cut_text(text='Some traditions believe in heaven, rebirth, or the immortality of the soul.', start=5, end=32))

파이썬 코드

def transform_texts(*texts, **kwargs):
    start = kwargs.get("start", 0)
    end = kwargs.get("end", 20)
    upper = kwargs.get("upper", False)
    result = []
    for text in texts:
        piece = text[start:end]
        if upper:
            piece = piece.upper()
        result.append(piece)
    return result

print(transform_texts('I will leave behind goodness through small acts of care.', 'With love and reflection, I can make today meaningful.', start=0, end=25, upper=True))

    'I will leave behind goodness through small acts of care.',
    'With love and reflection, I can make today meaningful.',
    'Today, I will live as a person whose kindness can remain after time passes.'
]

def final_mission(*texts, **kwargs):
    keyword = kwargs.get("keyword", "love")
    selected = list(filter(lambda text: keyword.lower() in text.lower(), texts))
    selected = sorted(selected, key=len)
    for index, text in enumerate(selected, start=1):
        print(index, text[:28], "...", text[-15:])
